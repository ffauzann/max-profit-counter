# Max Profit Counter & Array Shifter

This repository is for technical test purpose only. It's built with [Node.js](https://nodejs.org/).


## Installation

* Install [Node.js](https://nodejs.org/).
* Start the server. 
* For the first test:
```sh
$ npm run test-1
```
* For the second test:
```sh
$ npm run test-2
```
